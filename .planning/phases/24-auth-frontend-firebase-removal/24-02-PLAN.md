**All files using getIdToken pattern (AdminSettings, Extract, Title, Proration, Revenue, GhlPrep, Dashboard, MineralRights):**

Each file has this pattern:
```typescript
const { ..., getIdToken } = useAuth()
const authHeaders = useCallback(async (): Promise<Record<string, string>> => {
    const token = await getIdToken()
    ...
}, [getIdToken])
```

Change to:
```typescript
const { ..., getToken } = useAuth()
const authHeaders = useCallback((): Record<string, string> => {
    const token = getToken()

    if (token) headers['Authorization'] = `Bearer ${token}`
    return headers
}, [getToken])
```

Then in each file, change every `await authHeaders()` to `authHeaders()` (remove await since it's now sync). The `await` on a non-Promise is harmless but should be cleaned up.

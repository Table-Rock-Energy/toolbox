         ```typescript
           validateEnabled: boolean
           enrichEnabled: boolean
           onCleanUp: () => void
           onValidate: () => void
           onEnrich: () => void
           isProcessing: boolean    // true when ANY enrichment op running
           entryCount: number       // number of entries to process
         }
         ```
       - If no buttons enabled, return `null` (ENRICH-02: no broken buttons visible)
       - Three buttons conditionally rendered: Clean Up (Wand2 icon), Validate (MapPin icon), Enrich (Search icon)

       - ALL buttons disabled when `entryCount === 0`
       - Clean Up button: `bg-gradient-to-r from-tre-teal to-tre-teal/80 text-white` (primary action styling)
       - Validate and Enrich buttons: `border border-gray-300 text-gray-700` (secondary action styling)
       - When `isProcessing`, button text shows "Processing..." on the active button
       - Enrich button shows count: "Enrich (N)"

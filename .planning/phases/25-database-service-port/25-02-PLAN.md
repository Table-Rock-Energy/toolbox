Pattern for sync operations:
```python
def create_rrc_sync_job() -> str:
    session = get_sync_session()
    try:
        job = RRCDataSync(id=job_id, status="downloading_oil", ...)
        session.add(job)
        session.commit()
        return job_id
    except Exception:
        session.rollback()
        raise
    finally:
        session.close()
```

Also update `rrc_data_service.sync_to_database()` calls -- these use asyncio.run() inside the background thread, which creates a new event loop. Use `async_session_maker()` inside the new event loop.

    **4. Create openai_provider.py:**
    - Import `AsyncOpenAI` from `openai`
    - Create `parse_json_response(text: str) -> dict` function:
      - Try `json.loads(text)` directly
      - Try extracting from ```json ... ``` fence with `re.search(r"```(?:json)?\s*\n?(.*?)\n?```", text, re.DOTALL)`
      - Try extracting first `{ ... }` block with `re.search(r"\{.*\}", text, re.DOTALL)`
      - Raise `ValueError` if all fail
    - Create `OpenAIProvider` class:
      - `__init__`: lazy `_client: AsyncOpenAI | None = None`
      - `_get_client() -> AsyncOpenAI`: creates client with `base_url=settings.llm_api_base`, `api_key=settings.llm_api_key or "not-needed"`
      - `is_available() -> bool`: returns `settings.ai_provider == "lmstudio"`
      - `async def cleanup_entries(self, tool, entries, *, source_data=None, disconnect_check=None) -> list[ProposedChange]`:
        - Batch entries (batch_size=25)
        - For each batch, call `_get_client().chat.completions.create()` with:
          - `model=settings.llm_model`
          - system message = CLEANUP_PROMPTS.get(tool, CLEANUP_PROMPTS["extract"])
          - user message = formatted entries JSON with instructions to return JSON suggestions
          - `temperature=0.1`
        - Parse response via `parse_json_response()`
        - Convert suggestions to `ProposedChange` objects (source="ai_cleanup", authoritative=False)
        - Check `disconnect_check()` between batches
        - No rate limiting (local inference)
        - No batch delay (local inference)
      - `async def validate_entries(self, tool, entries) -> AiValidationResult`:
        - Uses TOOL_PROMPTS (validation prompts, not cleanup prompts) from prompts.py
        - Batches entries, calls LLM, parses JSON, returns AiValidationResult
        - This replaces `gemini_service.validate_entries()`
      - `async def verify_revenue_entries(self, entries, context=None) -> AiValidationResult`:
        - Uses REVENUE_VERIFY_PROMPT from prompts.py
        - Replaces `gemini_service.verify_revenue_entries()`

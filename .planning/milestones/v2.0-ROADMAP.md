# Milestone v2.0: Full On-Prem Migration

**Status:** SHIPPED 2026-03-25
**Phases:** 22-27
**Total Plans:** 13

## Overview

Removed all Google cloud dependencies (Firebase Auth, Firestore, GCS, Gemini) and migrated to fully on-prem stack: PostgreSQL + local filesystem + LM Studio for AI. JWT-based auth with bcrypt passwords replaces Firebase Auth. SQLAlchemy async sessions replace Firestore. OpenAI-compatible provider replaces Gemini.

## Milestone Summary

**Key Decisions:**
- PyJWT + pwdlib[bcrypt] for auth (FastAPI official recommendations, python-jose/passlib abandoned)
- OpenAI SDK for LM Studio (not litellm or lmstudio SDK)
- Alembic for migrations (not create_all)
- Remove Firestore entirely, no toggle — PostgreSQL is the only database
- Admin-only user creation (no self-registration)
- 24-hour JWT expiry, no refresh tokens (small internal team)
- Do NOT touch Dockerfile or docker-compose files

**Issues Resolved:**
- Admin user creation missing DB row insert (found during audit, fixed inline)
- GoogleCloud→ApiSettings endpoint rename conflicts (merge conflict resolution)

**Technical Debt Incurred:**
- VITE_FIREBASE_* ARGs still in Dockerfile (user said don't touch)
- JSON allowlist dual-path with PostgreSQL users table (is_user_admin reads JSON)
- check_user endpoint unauthenticated (by design for login flow)

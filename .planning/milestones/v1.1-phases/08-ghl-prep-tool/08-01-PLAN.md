Create `toolbox/backend/app/services/ghl_prep/transform_service.py`:
- `transform_csv(file_bytes: bytes, filename: str) -> TransformResult`
  1. Read CSV with pandas (`pd.read_csv(io.BytesIO(file_bytes))`). Handle encoding issues with `encoding="utf-8"` with fallback to `"latin-1"`.
  2. Identify text columns to title-case: Look for columns whose names contain "name", "city", "county", "territory", "address" (case-insensitive). Also title-case "First Name", "Last Name", "Full Name" if present. Do NOT title-case columns like "Email", "Phone", "State", "Zip".
  3. Title-case logic in helper function `title_case_name(value: str) -> str`:
     - If value is empty/NaN, return as-is
     - Use Python's `str.title()` as base
     - Post-process for special prefixes: Mc (McDonald not Mcdonald), Mac (MacArthur not Macarthur), O' (O'Brien not O'brien)
     - Preserve known suffixes that should stay uppercase: LLC, LP, LLP, Inc, Jr, Sr, II, III, IV
     - Preserve all-caps company indicators: if value contains "LLC", "LP", "INC", "CORP", "CO", "TRUST", "ESTATE" — still title-case but preserve the suffix abbreviation
     - Handle "MC DONALD" -> "McDonald", "O BRIEN" -> "O'Brien" patterns (space before prefix)
  4. Campaign extraction: Find column named "Campaigns" (case-insensitive). For each value:
     - If empty/NaN, leave as empty string
     - Try `json.loads(value)` — if it's a JSON array, extract first element's `unit_name` field
     - If JSON parsing fails, leave the raw value as-is (it may already be plain text)
     - Replace the Campaigns column value with the extracted plain text
  5. Phone mapping: Find column named "Phone 1" (case-insensitive). If found:
     - Add/overwrite a "Phone" column with the "Phone 1" values
     - Keep "Phone 1" column as-is (don't remove it)
  6. Contact Owner: Check if "Contact Owner" column exists (case-insensitive). If not, add it as empty string for all rows.
  7. Return TransformResult with all rows as list[dict] (using `df.to_dict(orient="records")`), with transformation counts.

Important: The transform must preserve ALL original columns. It modifies in-place and adds new columns. Column order should be: original columns in original order, then any new columns added (Phone, Contact Owner if they were added).

**AuthContext.tsx -- full rewrite (remove ALL Firebase imports):**

1. Define LocalUser interface at top of file:
```typescript
export interface LocalUser {
  email: string
  displayName: string | null
  photoURL: string | null
  id: string  // use email as stable ID
}
```

2. Update AuthContextType:
- Change `user: User | null` to `user: LocalUser | null`
- REMOVE `signInWithGoogle`
- Change `getIdToken: () => Promise<string | null>` to `getToken: () => string | null` (synchronous)
- Keep all other fields identical (userName, loading, isAuthorized, isAdmin, userRole, userScope, userTools, authError, backendReachable)

3. Remove ALL imports from firebase/auth and ../lib/firebase. Only import from react and ../utils/api.

4. signInWithEmail implementation:
- POST to `${API_BASE}/auth/login` with JSON `{email, password}`
- On non-ok response: parse JSON, throw Error with `data.detail || 'Login failed'`
- On success: store `data.access_token` in `localStorage.setItem('auth_token', ...)`
- Call `api.setAuthToken(data.access_token)`
- Build LocalUser from response: email from data.user.email, displayName computed as `[data.user.first_name, data.user.last_name].filter(Boolean).join(' ') || null`, photoURL: null, id: data.user.email
- Set all state: user, isAuthorized=true, isAdmin from data.user.is_admin, userRole from data.user.role, userScope from data.user.scope, userTools from data.user.tools, userName from localUser.displayName

5. signOut implementation:
- localStorage.removeItem('auth_token')
- api.clearAuthToken()
- Set user=null, isAuthorized=false, isAdmin=false, userRole=null, userScope=null, userTools=[], userName=null

6. getToken implementation (synchronous):
- Return `localStorage.getItem('auth_token')`

7. Mount/session-restore useEffect:
- Read token from localStorage('auth_token')
- If no token: setLoading(false), return
- Set api.setAuthToken(token)
- Probe backend health with retry loop (keep existing 3-attempt pattern)
- If not reachable: setBackendReachable(false), setLoading(false), return
- GET `${API_BASE}/auth/me` with Authorization Bearer header
- If response ok: parse UserProfile, build LocalUser, set all state (same as login)
- If response not ok (401/other): localStorage.removeItem('auth_token'), api.clearAuthToken(), setLoading(false)
- Register 401 handler via api.setUnauthorizedHandler: clear localStorage token, clear api token, set user=null, isAuthorized=false, authError='Your session has expired. Please sign in again.', return false (NO retry)

8. Remove checkAuthorization function entirely (authorization is now implicit in JWT -- if /me succeeds, user is authorized).

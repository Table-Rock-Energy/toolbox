## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | "Got it" (dismiss button on 403 modal) |
| Empty state heading | Not applicable -- no new empty states in this phase |
| Empty state body | Not applicable |
| Error state (403 delete) | **Heading:** "Unable to Delete" **Body:** "You can only delete jobs you created. Contact an admin if this job needs to be removed." |
| Destructive confirmation | Not applicable -- no new destructive actions (existing delete flow unchanged, just adds ownership check) |

**403 modal interaction:**
- Triggered when `DELETE /history/jobs/{id}` returns HTTP 403
- Uses existing `Modal` component at size `sm`
- Shows ShieldAlert icon (Lucide), heading, body text, single dismiss button
- Dismiss closes modal, job remains in list (no optimistic removal)
- No retry action -- the user cannot delete someone else's job
